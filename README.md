# C++ Todo List Manager

Un gestionnaire de tâches simple en ligne de commande (CLI) écrit en C++.

## Fonctionnalités
- Ajouter, modifier et supprimer des tâches.
- Afficher et trier les tâches.
- Sauvegarder et charger les tâches depuis le fichier `data/tasks.txt`.

## Compilation
Pour compiler le projet, exécutez la commande suivante à la racine :
```bash
g++ -I./include src/*.cpp -o todo_app
```

## Exécution
Une fois compilé, lancez l'application avec :
```bash
./todo_app
```
